# Тестовое задание: Институт охраны здоровья

Репозиторий с решением тестового задания (3 задачи) на позицию технического специалиста / системного аналитика.

## Структура

- `task1-web-schedule` — веб-утилита для просмотра расписания и записей пациентов.
- `task2-python-analytics` — Python-сервис для анализа загруженности врачей и неявок.
- `task3-apps-script-report` — скрипт Google Apps Script для ежедневного отчёта посещаемости.
